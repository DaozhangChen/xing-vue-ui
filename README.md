# Xing UI
这是一个基于Vue3的UI组件库，使用简单方便，可以通过npm、yarn、pnpm等包管理器进行安装。  
目前的组件有Switch、Button、Dialog、Tabs，组件设计简洁优雅，易于上手。
# 如何使用
```js
//使用npm
npm install xing-vue-ui

//使用yarn
yarn add xing-vue-ui

//使用pnpm
pnpm install xing-vue-ui
```
# 如何开发
克隆或下载github代码，之后使用
```js
//使用pnpm安装依赖
pnpm install

//编译库文件
pnpm build:lib
```
